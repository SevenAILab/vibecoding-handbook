# VibeCoding 实战手册

> 不会写代码，也能做产品。

这是一本写给非程序员的 AI 产品实战手册。

它不教你系统学习编程，而是教你如何用 AI 把一个真实想法推进成一个真实产品：定义需求、生成第一版、修改、验收、上线、观察、收钱、迭代。

## 适合谁

- 有产品想法，但不会写代码的人
- 品牌、运营、内容、设计、咨询、教育等非技术背景的人
- 想用 AI 做网站、工具、小程序、轻 SaaS 的独立创作者
- 已经会用 AI 工具，但不知道怎么把想法推进到上线的人
